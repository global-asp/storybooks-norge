Lukk

मैं इसके साथ दौड़ सकता हूँ।

Jeg kan løpe med den.

Eg kan springa med han.

मैं इसके साथ कूद सकता हूँ।

Jeg kan hoppe med den.

Eg kan hoppa med han.

मैं इसके साथ नाच सकता हूँ।

Jeg kan danse med den.

Eg kan dansa med han.

मैं इसके साथ तैर सकता हूँ।

Jeg kan svømme med den.

Eg kan symja med han.

मैं इसके साथ रस्सी कूद सकती हूँ।

Jeg kan hoppe tau med den.

Eg kan hoppa tau med han.

मैं इसके साथ ठोकर मार सकता हूँ।

Jeg kan sparke med den.

Eg kan sparka med han.

मैं इसके साथ भाग सकता हूँ।

Jeg kan stikke av med den.

Eg kan stikka av med han.

लेकिन मैं इसके साथ उड़ नहीं सकता।

Men jeg kan aldri fly med den.

Men eg kan aldri flyga med han.
