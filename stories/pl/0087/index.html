Lubię czytać!

Jeg liker å lese.

Eg likar å lesa.

Komu mogę poczytać?

Hvem kan jeg lese for?

Kven kan eg lesa for?

Moja siostra już śpi.

Søsteren min sover.

Syster mi søv.

Komu mogę poczytać?

Hvem kan jeg lese for?

Kven kan eg lesa for?

Mama i babcia są zajęte.

Mamma og bestemor har ikke tid.

Mamma og bestemor har ikkje tid.

Komu mogę poczytać?

Hvem kan jeg lese for?

Kven kan eg lesa for?

Tata i dziadek są zajęci.

Pappa og bestefar har ikke tid.

Pappa og bestefar har ikkje tid.

Komu mogę poczytać? Mogę poczytać sobie sam!

Hvem kan jeg lese for? Jeg kan lese for meg selv!

Kven kan eg lesa for? Eg kan lesa for meg sjølv!
