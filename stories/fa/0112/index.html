بَدَنِ مَن Kroppen min Kroppen min

Persian

Nivå 1

مَن می تَوانَم با آن بِدَوَم.

Jeg kan løpe med den.

Eg kan laupa med han.

مَن می تَوانَم با آن بِپَرَم.

Jeg kan hoppe med den.

Eg kan hoppa med han.

مَن می تَوانَم با آن بِرَقصَم.

Jeg kan danse med den.

Eg kan dansa med han.

مَن می تَوانَم با آن شِنا کُنَم.

Jeg kan svømme med den.

Eg kan symja med han.

مَن می تَوانَم با آن بِه این طَرَف وَآن طَرَف بِپَرَم.

Jeg kan hoppe med den.

Eg kan hoppa med han.

مَن می تَوانَم با آن لَگَد بِزَنَم

Jeg kan sparke med den.

Eg kan sparka med han.

مَن می توانَم با آن فَرار کُنَم.

Jeg kan stikke av med den.

Eg kan stikka av med han.

وَلی هَرگِز نِمی تَوانَم با آن پَرواز کُنَم.

Men jeg kan aldri fly med den.

Men eg kan aldri flyga med han.
